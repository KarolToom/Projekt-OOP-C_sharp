Założeniem projektu jest utworzenie aplikacji w języku C\#, korzystając z platformy NET. Aplikacja jest typy Windows Forms, wykorzystuję relacyjną baze danych Microsfot SQL Server (LocalDB) do trwałego przechowywania informacji.
Służy ona do zarządzania bazą danych pracowników w przykładowym przedsiębiorstwie. 
Projekt ma na celu zastosowania metod Programowania Obiektowego (OOP) do obsługi wynagrodzeń dla różnych specjalistów obecnych w firmie.
